# projet1_g4

## Name
Gambling city

## Description
Un jeu de casino se composant de plusieurs mini-jeux : Blackjack, Roulette, Baccara, Machine a sous, Gobelets, Boite mystère ainsi que la possibilité deprendre part à des ventes aux enchères artistiques.

## Installation
Utilise Python 3.12
Nécessite pygame 2.6, sqlite3, random, time
